# MASA-OpenMP

<p align="justify">
The <b>MASA-OpenMP extension</b> is used with the <a href="https://github.com/edanssandes/MASA-Core">MASA architecture</a> to align DNA sequences of unrestricted size with the Smith-Waterman algorithm combined with Myers-Miller. It uses the OpenMP framework to accelerate the computation time in CPU. 
</p>
